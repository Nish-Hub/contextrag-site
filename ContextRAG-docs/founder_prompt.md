Project: ContextRAG

Mission:
Give LLMs the Context They Need.

ContextRAG is a Context Extension Platform that enhances Large Language Model interactions by retrieving project knowledge before sending prompts to the LLM.

Instead of querying LLMs directly, ContextRAG retrieves relevant context from a vector database and constructs an augmented prompt.

Core Architecture:

• Multi-LLM Custom UI  
• Java RAG Service (orchestration layer)  
• Vector Database (Weaviate)  
• Embedding Models  
• Kafka Event Pipeline  
• Conversation Summarisation Layer

Knowledge Sources:

• uploaded documents  
• project architecture notes  
• developer conversations  
• code snippets
